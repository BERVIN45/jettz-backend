# Jettz Backend 🚀

Spring Boot backend for handling enquiry form submissions.

## 📌 Features
- Save enquiry data to database
- REST API integration
- CORS enabled for frontend

## 🛠 Tech Stack
- Java
- Spring Boot
- MySQL
- JPA / Hibernate

## 🔗 API Endpoint
POST /api/enquiry
